Description


A table showing cohort characteristics (covariates). These characteristics are captured on or before the cohort start date.


Options


You can select multiple target cohorts in the drop down to see cohort characteristics from different targets side-by-side in the same table.

Select the database drop down to view characteristics in the target cohort(s).

Select the generate report button to populate and display a table in the Table box.

Uncheck the Mean and Count boxes to display a table with average values or counts by covariate.

Once the table is viewable select the download button to create a .csv file with the information contained in the table.

Select the next text or an increaseing number to review the information contained on subsequent pages of the table.

What to look for

  • Are the characteristics of the cohort as expected? For example, do people have the expected comorbidities?
  • Do the characteristics of the cohort differ by database?
